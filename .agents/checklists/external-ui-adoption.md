# External UI Adoption Checklist

- External libraries are treated as references or dependency candidates, not source-of-truth owners.
- Public API does not expose external library types or Tailwind/shadcn structures.
- Copied or ported code has license and third-party notice review.
- Reduced motion, keyboard, focus, and accessibility behavior is preserved.
- `docs/EXTERNAL_UI_ADOPTION.md` and `THIRD_PARTY_NOTICES.md` stay synchronized when adoption status changes.
